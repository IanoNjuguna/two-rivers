# Neue Machina Font

This is a placeholder. To use the Neue Machina font:

1. Download or purchase Neue Machina font files from:
   - [purchase](https://pangrampangram.com/) (official)
   - Or your preferred font source

2. Place the following files in this directory:
   - NeueMachina-Light.otf (weight: 300)
   - NeueMachina-Regular.otf (weight: 400)
   - NeueMachina-Ultrabold.otf (weight: 900)

3. The layout.tsx is already configured to load these fonts.
